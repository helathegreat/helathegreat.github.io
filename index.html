如何部署hexo

0 前期准备

1.安装cnpm，nodejs,安装hexo

1 本地环境建立

1.新建文件夹 hexo init
2.hexo new(n) 写新 –自动放到source文件夹下
3.hexo server(s) 本地浏览
4.hexo generate(g) 生成

2 git部署

1.仓库命名 name.github.io
2.git 部署插件 cnpm install –save hexo-deployer-git
3.编辑configyml
— deployment
—-1. type git
2. repo:webpage
3. branch:master
4.hexo d 进行部署

3 风格变换

1.git clone 到theme
2.config中更换名称
